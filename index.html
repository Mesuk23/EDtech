Bootstrap is a CSS framework in which we can find some ready made product. So We can work easily and it is time saving for us. But there are limited option for make changes.
But on the other hand , Tailwind is an utility based CSS framework. In which we can find so many option to customize our website. In tailwind we have so many option to make changes

Flexbox is one dimentional layout in which we can control only one row. But Grid is two dimentional and we can control both row and column. And also we can customize the size of the columns and rows.

CSS box model is the wraping of the HTML elements ,where the main content are bound by padding, border and margin in each and every side of it and make a structure like a box.

Semantic tags means the meaningfull tags in the HTML elements , by which the developer and browser can understand the website layout easily. For example: using aside , main, section, footer , header tags except of using div tag everywhere.
